This dashboard has been developed to support the systematic monitoring and coordination of the BRIDGE project across all partner institutions. It provides a centralised view of work package progress, milestones, deliverables, and bootcamp activities, enabling timely tracking and informed decision-making.
Purpose
The dashboard is intended to:
Ensure transparency in project implementation
Track progress against approved timelines and deliverables
Support early identification of risks and delays
Facilitate internal reporting and Erasmus+ compliance
Data Updates
Partners are requested to:
Regularly update information related to their assigned deliverables and activities
Ensure that all entries are accurate and up to date
Provide supporting evidence links where applicable
Flag delays or risks at the earliest stage
Status Definitions
On Track: Activity progressing as planned
Due Soon: Activity approaching deadline
Overdue: Deadline has passed and action is required
Usage
The dashboard should be used as a shared operational tool during coordination meetings and internal reviews. It is not only for reporting purposes but also to support proactive project management across the consortium.
